Links

 several of my favourite links, very useful when identifying some piece of hardware or making it work;-). follow and maybe you find what you are looking for..

Plasma online

 great hardware identification using huge database of information and chip - photos, interface specifications, tweaks, pc - troubleshooting help

BIOS Central

 POST diagnostics, beeps, errorcodes, technical reference, upgrading notes, ..

Datasheet Archive

 datasheet and application note database, PDF, circuits. indispensable for mysterious pieces of hardware marked with the only number;-)

Nu2.nu

 everything about booting not only from CDs, a lot of utilities about it, boot sectors for various OSes

Pinouts.ru

 easy domain name - evident purpose. lot of pinouts, cable schemes and connector layouts from the basic ones to the rare or non-standard ones

PITT' s Electronics Home

pinouts, schemes, layouts, plans for various electronic devices from LEDs and easy displays to computer interface.

Nobbi.com

 network - monitoring software for mobile phones, cable pinouts, a lot of information, BTS photographs, ..

Handy-sammler.de

 incredible collection of mobile phones here. what to say more?

WeOCR server

 online OCR service using tesseract ocr - engine, useful for quick recognition anywhere (online;-) )
